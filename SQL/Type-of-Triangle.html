SQL Hackerrank Type of Triangle

更新于 2020-03-04 分类于 SQL 阅读次数：
本文字数： 2k 阅读时长 ≈ 2 分钟

Problem

Write a query identifying the type of each record in the TRIANGLES table using its three side lengths. Output one of the following statements for each record in the table:

  • Equilateral: It’s a triangle with 3 sides of equal length.
  • Isosceles: It’s a triangle with 2 sides of equal length.
  • Scalene: It’s a triangle with 1 sides of differing lengths.
  • Not A Triangle: The given values of A, B, and C don’t form a triangle.

Input Format

The TRIANGLES table is described as follows:

Column Type
A Integer
B Integer
C Integer

Each row in the table denotes the lengths of each of a triangle’s three sides.

Sample Input

A B C
20 20 23
20 20 20
20 21 22
13 14 30

Sample Output

Explanation

  • Values in the tuple $(20,20,23)$ form an Isosceles triangle, because $A=B$.
  • Values in the tuple $(20,20,20)$ form an Equilateral triangle, because $A=B=C$.
  • Values in the tuple $(20,21,22)$ form a Scalene triangle, because $A\ne B\ne C$ and $A\ne C$.
  • Values in the tuple $(13,14),30$ cannot form a triangle because the combined value of sides A and B is not larger than that of side C.

Idea

This is a math problem. If the tuple $(A,B,C)$ form a triagnle, we must have
$$(A+B-C)(A+C-B)(B+C-A)>0.$$To see this, suppose $A$ is the largest side, then $A+B-C>0$ and $A+C-B>0$. If we get a triagnle, then we must have $B+C-A>0$. Now we are ready for the answer.

  • Use CASE and WHEN (analogous to if and else if).
  • First, we consider the case if it does not form a triangle. Then we mush have (A+B-C)*(A+C-B)*(B+C-A) <= 0.
  • Second, we consider the case all sides are equal.
  • Then, only two sides are equal. Note that this is if and else if, this only happens if (A-B)*(B-C)*(C-A) = 0.
  • Finally, the rest are 'Scalene'.
